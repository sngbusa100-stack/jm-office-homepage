# 상담 저장 실패 폴백 운영 절차

텔레그램 알림에 `접수 저장 실패 — 이 알림이 유일한 기록입니다`가 표시된 경우에만 적용한다.

1. 알림의 접수번호·연락처·상담 내용을 로컬 행정사 프로그램의 홈페이지 문의 대기함 또는 사건 접수 폼에 즉시 수동 등록한다.
2. 같은 접수번호로 로컬 저장됐는지 다시 조회하고, 연락처와 상담 내용이 일치하는지 확인한다.
3. 행정사 프로그램의 진단·증거 체크 결과는 자동 발송하지 않고 행정사가 검토한다.
4. 로컬 등록과 백업을 확인한 뒤 텔레그램의 개인정보 포함 메시지를 삭제한다.
5. 저장 장애 원인을 확인하고 복구한 뒤 `/admin`과 connector 목록에서 신규 접수가 다시 보이는지 시험한다.

일반 `새 상담 접수` 알림에는 개인정보가 없으므로 이 절차의 대상이 아니다.
